### Scenario 1: Writing New Tool Content
```
Step 1: Open SEO_CONTENT_STRATEGY.md
        └─→ Find your tool in the list
        └─→ Review the content outline

Step 2: Open KEYWORD_RESEARCH_DATABASE.md
        └─→ Find keywords for your tool
        └─→ Note primary + secondary keywords

Step 3: Write content following the 5-section structure
        └─→ Intro (150-200 words)
        └─→ How It Works (150-200 words)
        └─→ Use Cases (150-200 words)
        └─→ FAQ (250-300 words)
        └─→ Benefits (100-150 words)

Step 4: Optimize using keyword database
        └─→ Add primary keyword to H2 + first paragraph
        └─→ Integrate secondary keywords naturally
        └─→ Use long-tail keywords in FAQs

Step 5: Check pre-publish checklist (in Calendar doc)
        └─→ All formatting correct?
        └─→ Internal links added?
        └─→ Meta description written?

Step 6: Publish and track in Calendar
```
